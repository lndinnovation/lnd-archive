Second Nature
AI 세일즈 트레이닝 플랫폼 (영상형 AI 코치 "Jenny")
▼
회사명Second Nature
홈페이지secondnature.ai
핵심 솔루션"Jenny"라는 영상형 AI 세일즈 코치 아바타와의 롤플레잉을 통해 세일즈 역량을 향상하는 B2B 세일즈 트레이닝 플랫폼
주요 기능
다양한 시뮬레이션 유형
  • 1:1, 그룹 등 다양한 상황에서 AI 아바타와 롤플레잉 가능 (30개 이상의 언어 보유)
  • 프레젠테이션을 기반으로 롤플레잉하는 Slide Presentation Simulation 가능
  • Jenny에게 발표하고, 정보를 전달하며, Jenny가 질문을 던지고 즉시 피드백을 주는 프레젠테이션 롤플레잉
교육 컨텐츠 연동
  • 교육 콘텐츠, 퀴즈, 프레젠테이션 자료, 롤플레잉을 한 플랫폼에서 학습 가능
AI 롤플레잉 저작 도구
  • 자유 형식으로 시나리오를 설명하거나 콘텐츠(PDF, 녹음 등)를 업로드하면 AI가 롤플레잉 제작 등을 자동 처리
  • 시나리오 컨텍스트, 페르소나 배경, 대화 구조, 다룰 주제(가중치 점수 포함), 스타일 평가 매개변수 및 합격 기준 정의
  • 지정된 영업 방법론 안에서 생성 가능
AI 피드백 제공 (개인/조직 리포트)
  • 개인 리포트: 커뮤니케이션, 제품 설명 등 개인 맞춤형 피드백 제공
  • 평가 기준: 70% 콘텐츠 정확성(범위 제공) + 30% 커뮤니케이션 스타일(속도, 명확성, 활력, 불필요한 단어 사용 등) — 비율 조정 및 항목 변경 가능
  • 조직 리포트: 진행율, 강약점 히트맵 등을 보여주는 대시보드 제공 (팀별 비교 분석 등)
  • 롤플레이 시간이 너무 짧을 경우, 롤플레잉 종료 전 피드백 제공이 어렵다는 메시지 노출
시스템 연동LMS, CMS 등의 플랫폼과 시스템 연동 가능

Khanmigo (칸미고)

홈페이지khanacademy.org · khanmigo.ai
상품명칸미고 (Khanmigo) — GPT-4 기반 AI 학습 솔루션
타깃교사, 학생, 부모 (타깃에 따른 솔루션 상이)
주요 기능
학생용 AI 툴
  • Tutor mode: 소크라테스식 문답법 적용 — 질문에 바로 답을 하지 않고, "이 문제에서 분자는 뭐라고 생각하니?" 등 생각을 유도하는 질문과 힌트를 제공하여 학습자가 스스로 답을 찾도록 함
  • 학습자 데이터와 연동되어 어떤 부분을 공부하고 있는지 확인 가능, 영상 컨텐츠 참고 가능
  • Write coach: 브레인스토밍을 통한 스토리 구성, 에세이 피드백 — 4단계(과제 이해 → 개요 작성 → 초안 작성 → 피드백 및 수정). 대신 써주는 것이 아니라 직접 쓸 수 있도록 가이드/피드백/동기부여 제공
  • Debate practice: 토론 연습
  • Coding support: 프로그래밍 학습 시 실시간 피드백 제공
  • College counseling: 대학 입학, 진로 계획 등 미래 계획 도움
  • Refresh: 복습을 위한 퀴즈 생성
  • Chat: 역사적 인물/문학 속 인물 등 가상의 인물과의 채팅
교사용 AI 툴 (25개)
  • 퀴즈 생성, 예시 생성, 뉴스레터 생성 등 교사를 위한 학습 지원 툴 제공
데이터 리포트
  • 대화 요약·인사이트, 채팅 메시지 수, 학생별 활동량, 진도, 학습시간
금액
  • 학습자/학부모: 월 $4 또는 연 $44
  • 교사: 무료
참고: 칸미고 교수용 툴 상세
영역이름내용
Create Blooket Generator 퀴즈 생성 — 학년, 주제, 목적 등을 입력하면 자동으로 문제 생성. Blooket과 연계되어 퀴즈 생성 후 Blooket 페이지로 연결

Create Informational Text 정보 기반의 아티클 생성 — 학년, 주제, 분량 입력 (100~500자 이내)
Create Letter of Recommendation 학생 추천서 생성 — 학생 이름, 목적, 학생과의 관계, 성과, 성장 가능성 등 입력
Create Report Card Comments 개별 피드백 생성 — 학생 이름, 학년, 강조하고 싶은 부분, 개선해야 되는 부분 입력
Create Rubric Generator 평가 기준 생성 — 학습 목표, 과제 내용, 평가 기준, 레벨 입력 → 레벨별 평가 기준 생성
Plan Discussion Prompts 토론 주제 생성 — 주제와 학년 입력
Plan Exit Ticket 주관식 문제 생성 — 학년 및 주제 입력
Plan Learning Objective(s) 학습 목표 생성 — 학년 및 주제 입력
Plan Lesson Hook 학습 분위기를 올리기 위한 액티비티 생성 — 학년 및 주제 입력
Plan Lesson Plan 학습 계획 생성 (활동 액티비티 중심) — 학습 주제, 목표, 내용, 포멧, 디테일 정도 입력
Differentiate Chunk Text 긴 문장의 내용을 단락화 (소주제 생성)
Differentiate Leveler 1,500자 이내 글을 각 학년의 수준에 맞게 조정
Differentiate Make it Relevant! 학생/반의 상황을 고려한 학습 목표 생성 — 예: 축구를 좋아하는 학생 → 축구 선수의 득점 데이터를 이용해 평균변화율 계산하기
Differentiate Real World Context Generator 실사례 생성 — 학년 및 주제 입력
Differentiate Text Rewriter 원문을 원하는 방향으로 재작성해주는 툴
Support IEP Assistant 학생 개별 지도 계획 생성 — 학습 주제, 직면한 문제 및 상황, 현재 강점 작성 → 현재 상황, 구체적 제언, 목표 등 생성
Support SMART Goal Writer SMART 기반 목표 생성 — 이름, 최종 목표, 원하는 결과물 입력
Learn Refresh My Knowledge 대화 기반 복습 — 학년 및 주제를 입력하면 퀴즈 등을 제시하며 대화 기반 복습

특장점
강의–문답 통합 단일 인터페이스
  • 음성 강의(텍스트 동기 노출)와 소크라틱 문답이 별도 단계가 아니라 하나의 채팅창 안에서 연속적으로 진행
'손들기' 버튼을 통한 중간 질문 기능
  • 강의 중 궁금한 점이 생기면 하단 손들기 버튼 클릭 → AI가 무엇이 궁금하냐고 물어보면서 질문 모드로 변경
논증 구조 기반의 문답
  • 학습자의 답변을 주장–이유–근거–가정으로 분석하여, 구조적 빈틈을 짚는 질문과 피드백으로 사고력 향상
뉴로심볼릭 AI 엔진 활용
  • 자유롭게 쓰인 문장을 잘 이해하는 AI(뉴럴) + 규칙·논리·지식구조 기반 AI(심볼릭)를 결합해, 논증 구조에 따라 체계적으로 문답
